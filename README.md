# Personal RAG Chatbot

This is my personal RAG learning project.

The goal is to upload personal PDF files, convert them into chunks, store them in a vector database, and ask questions using OpenAI.

## Current Stage

- Google Colab notebook
- PDF loading
- Text chunking
- OpenAI embeddings
- Chroma vector DB
- Retrieval-based question answering

## Important

Private PDFs and API keys are not stored in this repository.
